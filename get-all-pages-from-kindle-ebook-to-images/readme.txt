This script makes one screenshot for every
page in a Kindle ebook and saves them to a
specific folder. Then, you can merge all
images into a pdf, for example.

You must have Kindle for PC installed.

Get the mouse positions using the script 
show_mouse_position.py

Libs:
- pyautogui (execute $ pip install pyautogui)
